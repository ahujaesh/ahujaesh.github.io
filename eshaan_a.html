Biography

I'm Eshaan Ahuja, born in Toronto, Canada, and currently residing in The Woodlands, Texas. I find joy in coding, software development, and building and exploring the world of robotics. When not indulging in these tech-related pursuits, I'm an avid biker and runner, relishing the thrill of the outdoors. I also love reading and learning about new things. In addition to this, I actively participate in FTC (FIRST Tech Challenge) and serve as the robotics lead for my DI (Destination Imagination) team. In the FTC realm, I am a programmer, actively learning and refining my coding skills to contribute effectively to the team's success. You can read more about us and what we do here.

As I continue my journey, I aspire to turn my dreams into reality and make a mark in the tech industry.

My GitHub

GitHub is a developer platform that allows developers to create, store, manage, and share their code. I'm a small developer there and have some decent stats.

A GitHub Contributing Graph shows your open source contributions on GitHub over the last year and is available on any GitHub user's page. GitHub user Platane made a fun twist on this by making it into a game of Snake, in which the snake has to eat the days in order of their contribution weight. Unfortunately though, due to the randomness of my contributions, the graph isn't perfect and sometimes has to deviate from the correct color to get all the blocks. You can view my GitHub Contribution Graph, without the snake, here.

Key:

- The lighter the green, the less contributions were on that day. (Example image: )
- If the green is darker, there were more contributions. (Example image: )
- If the square is white, there were no contributions on that day.
